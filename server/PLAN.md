# SHEild Backend Implementation Plan

- Backend: `/server` folder inside the SHEild project
- Frontend: existing `/src` folder, wired to real API

## Execution Order
1. Create server/ directory structure + package.json
2. Config files (db, cloudinary, socket)
3. Models
4. Middleware
5. Services
6. Controllers
7. Routes
8. app.js + server.js
9. .env template
10. Frontend: update stores, wire components
